Bronze лҚ°мқҙн„°лҘј л°”лЎң м“°л©ҙ мғқкё°лҠ” мқј#

2нҺё м—җм„ң Bronzeм—җ мӣҗліёмқ„ мһҲлҠ” к·ёлҢҖлЎң м Ғмһ¬н–ҲлӢӨ. ліҖнҷҳ м—Ҷмқҙ. к·ё мӣҗм№ҷмқҖ л§һлӢӨ. л¬ём ңлҠ” Bronze лҚ°мқҙн„°к°Җ л¶„м„қм—җ м“ё мҲҳ мһҲлҠ” мғҒнғңк°Җ м•„лӢҲлқјлҠ” кІғмқҙлӢӨ.

jaffle_shopмқҳ bronze.ordersлҘј ліҙмһҗ. order_date м»¬лҹјмқҙ VARCHARлЎң л“Өм–ҙмҷҖ мһҲлӢӨ. лӮ м§ң н•ЁмҲҳлҘј м“ё мҲҳ м—ҶлӢӨ. status м»¬лҹјм—җлҠ” returned, return_pending, completed, placed, shippedк°Җ м„һм—¬ мһҲлҠ”лҚ°, м–ҙлҠҗ к°’мқҙ мөңмў… мғҒнғңмқём§Җ мҠӨнӮӨл§Ҳл§Ң лҙҗм„ңлҠ” лӘЁлҘёлӢӨ.

bronze.paymentsмқҳ amount м»¬лҹјмқҖ м„јнҠё лӢЁмң„ м •мҲҳлӢӨ. лӢ¬лҹ¬лЎң л°”кҫёл Өл©ҙ 100мңјлЎң лӮҳлҲ м•ј н•ңлӢӨ. мқҙкұё л¶„м„қн•  л•Ңл§ҲлӢӨ л§ӨлІҲ лӮҳлҲ„лҠ” кұҙ мӢӨмҲҳлҘј л¶ҖлҘҙлҠ” кө¬мЎ°лӢӨ.

SilverлҠ” мқҙлҹ° кІғл“Өмқ„ н•ң лІҲм—җ м •лҰ¬н•ҳлҠ” л Ҳмқҙм–ҙлӢӨ. нғҖмһ…мқ„ л§һм¶”кі , м»¬лҹјлӘ…мқ„ нҶөмқјн•ҳкі , лӢЁмң„лҘј ліҖнҷҳн•ңлӢӨ. л№„мҰҲлӢҲмҠӨ лЎңм§ҒмқҖ м•„м§Ғ л„Јм§Җ м•ҠлҠ”лӢӨ. “л¶„м„қм—җ м“ё мҲҳ мһҲлҠ” к№ЁлҒ—н•ң мғҒнғң"лҘј л§Ңл“ңлҠ” кІҢ Silverмқҳ м—ӯн• мқҙлӢӨ.

Silverм—җм„ң н•ҳлҠ” мқј, м•Ҳ н•ҳлҠ” мқј#

кІҪкі„лҘј м§ҖнӮӨлҠ” кІҢ мӨ‘мҡ”н•ҳлӢӨ. Silverм—җм„ң л№„мҰҲлӢҲмҠӨ лЎңм§Ғмқ„ л„Јкё° мӢңмһ‘н•ҳл©ҙ BronzeмҷҖ SilverлҘј лӮҳлҲҲ мқҳлҜёк°Җ мӮ¬лқјм§„лӢӨ.

Silverм—җм„ң н•ҳлҠ” мқј:

  • нғҖмһ… мәҗмҠӨнҢ… - VARCHARлҘј DATE, INTEGERлҘј DECIMALлЎң
  • м»¬лҹјлӘ… н‘ңмӨҖнҷ” - user_idмҷҖ userIdлҘј user_idлЎң нҶөмқј
  • лӢЁмң„ ліҖнҷҳ - м„јнҠёлҘј лӢ¬лҹ¬лЎң, л°ҖлҰ¬мҙҲлҘј мҙҲлЎң
  • мӨ‘ліө м ңкұ° - к°ҷмқҖ л ҲмҪ”л“ңк°Җ л‘җ лІҲ м Ғмһ¬лҗң кІҪмҡ°
  • NULL мІҳлҰ¬ - л№Ҳ л¬ёмһҗм—ҙмқ„ NULLлЎң нҶөмқј

Silverм—җм„ң м•Ҳ н•ҳлҠ” мқј:

  • KPI кі„мӮ° - л§Өм¶ң, л§Ҳм§„мңЁ к°ҷмқҖ л№„мҰҲлӢҲмҠӨ м§Җн‘ң
  • н…Ңмқҙлё” мЎ°мқё - мЈјл¬ёкіј кі к°қмқ„ н•©міҗм„ң н•ҳлӮҳмқҳ л·°лЎң л§Ңл“ңлҠ” кІғ
  • м§‘кі„ - GROUP BYлЎң мҡ”м•Ҫн•ҳлҠ” кІғ

мЎ°мқёкіј м§‘кі„лҠ” Goldмқҳ лӘ«мқҙлӢӨ. SilverлҠ” к°ңлі„ н…Ңмқҙлё” лӢЁмң„лЎң м •м ңл§Ң н•ңлӢӨ.

dbtк°Җ н•„мҡ”н•ң мқҙмң #

1нҺё м—җм„ң dbtлҘј лҸ„кө¬лЎң мҶҢк°ңн–ҲлӢӨ. мҷң SQL нҢҢмқјмқ„ м§Ғм ‘ мӢӨн–үн•ҳм§Җ м•Ҡкі  dbtлҘј м“°лҠ”к°Җ.

SQL нҢҢмқјмқ„ н•ҳлӮҳм”© мӢӨн–үн•ҳл©ҙ мІҳмқҢм—” л¬ём ңк°Җ м—ҶлӢӨ. Silver н…Ңмқҙлё”мқҙ 5к°ң, 10к°ңлЎң лҠҳм–ҙлӮҳл©ҙ мғҒнҷ©мқҙ лӢ¬лқјм§„лӢӨ. м–ҙл–Ө н…Ңмқҙлё”мқҙ м–ҙл–Ө Bronze н…Ңмқҙлё”м—җ мқҳмЎҙн•ҳлҠ”м§Җ, м–ҙл–Ө мҲңм„ңлЎң мӢӨн–үн•ҙм•ј н•ҳлҠ”м§Җ, л§Ҳм§Җл§ү мӢӨн–үмқҙ м–ём ңмқём§Җ м¶”м Ғмқҙ м•Ҳ лҗңлӢӨ.

dbtлҠ” мқҙкұё н•ҙкІ°н•ңлӢӨ. SQL нҢҢмқј н•ҳлӮҳк°Җ н•ҳлӮҳмқҳ лӘЁлҚёмқҙлӢӨ. ref() н•ЁмҲҳлЎң лӘЁлҚё к°„ мқҳмЎҙ кҙҖкі„лҘј м„ м–ён•ҳл©ҙ dbtк°Җ мӢӨн–ү мҲңм„ңлҘј м•Ңм•„м„ң м •н•ңлӢӨ. ліҖнҷҳ лЎңм§Ғмқҙ SQL нҢҢмқјм—җ лӮЁмңјлӢҲ GitмңјлЎң мқҙл Ҙ м¶”м ҒлҸ„ лҗңлӢӨ.

Silver лӘЁлҚё мһ‘м„ұ#

dbtм—җм„ңлҠ” models/staging/ л””л үнҶ лҰ¬м—җ Silver л Ҳмқҙм–ҙ лӘЁлҚёмқ„ л‘”лӢӨ. stg_ м ‘л‘җм–ҙк°Җ staging(=Silver)мқ„ лң»н•ңлӢӨ.

stg_orders#

%%writefile jaffle_shop/models/staging/stg_orders.sql
with source as (
    select * from bronze.orders
),

cleaned as (
    select
        id as order_id,
        user_id as customer_id,
        cast(order_date as date) as order_date,
        status
    from source
)

select * from cleaned

Bronzeмқҳ idлҘј order_idлЎң л°”кҝЁлӢӨ. м—¬лҹ¬ н…Ңмқҙлё”мқ„ мЎ°мқён•  л•Ң idл§ҢмңјлЎңлҠ” м–ҙлҠҗ н…Ңмқҙлё”мқҳ IDмқём§Җ м•Ң мҲҳ м—ҶмңјлӢҲк№Ң. user_idлҸ„ customer_idлЎң л°”кҝ”м„ң мқҳлҜёлҘј лӘ…нҷ•нһҲ н–ҲлӢӨ. order_dateлҘј DATEлЎң мәҗмҠӨнҢ…н–ҲлӢӨ.

CTE нҢЁн„ҙ#

мң„ SQLм—җм„ң л°ҳліөм ҒмңјлЎң м“°мқё нҢЁн„ҙмқҙ мһҲлӢӨ. with source as (...), cleaned as (...) select * from cleaned. dbt м»Өл®ӨлӢҲнӢ°м—җм„ң л„җлҰ¬ м“°мқҙлҠ” CTE(Common Table Expression) нҢЁн„ҙмқҙлӢӨ.

with source as (
    -- 1лӢЁкі„: Bronzeм—җм„ң мӣҗліёмқ„ к°Җм ёмҳЁлӢӨ
    select * from bronze.orders
),

cleaned as (
    -- 2лӢЁкі„: м •м ң лЎңм§Ғмқ„ м Ғмҡ©н•ңлӢӨ
    select
        id as order_id,
        cast(order_date as date) as order_date
    from source
)

-- 3лӢЁкі„: мөңмў… кІ°кіјлҘј л°ҳнҷҳн•ңлӢӨ
select * from cleaned

source вҶ’ cleaned вҶ’ select. к°Ғ лӢЁкі„к°Җ лӯҳ н•ҳлҠ”м§Җ мқҙлҰ„м—җм„ң мқҪнһҢлӢӨ. м •м ң лЎңм§Ғмқҙ ліөмһЎн•ҙм§Җл©ҙ CTEлҘј м¶”к°Җн•ҳл©ҙ лҗңлӢӨ. renamed, filtered, deduplicated к°ҷмқҖ мқҙлҰ„мңјлЎң лӢЁкі„лҘј лӮҳлҲ„лҠ” нҢҖлҸ„ мһҲлӢӨ.

мӨ‘ліө м ңкұ° нҢЁн„ҙ#

Bronzeм—җ к°ҷмқҖ л ҲмҪ”л“ңк°Җ л‘җ лІҲ л“Өм–ҙмҳӨлҠ” кІҪмҡ°к°Җ мһҲлӢӨ. мҶҢмҠӨ мӢңмҠӨн…ңм—җм„ң лҚ°мқҙн„°лҘј лӢӨмӢң ліҙлғҲкұ°лӮҳ, мҰқл¶„ м Ғмһ¬ лЎңм§Ғм—җ лІ„к·ёк°Җ мһҲм—Ҳкұ°лӮҳ. Silverм—җм„ң мқҙкұё мһЎм•„м•ј н•ңлӢӨ.

with source as (
    select * from bronze.orders
),

deduplicated as (
    select
        *,
        row_number() over (
            partition by id
            order by _loaded_at desc
        ) as row_num
    from source
),

cleaned as (
    select
        id as order_id,
        user_id as customer_id,
        cast(order_date as date) as order_date,
        status
    from deduplicated
    where row_num = 1
)

select * from cleaned

row_number()лЎң к°ҷмқҖ idк°Җ м—¬лҹ¬ к°ң мһҲмңјл©ҙ к°ҖмһҘ мөңк·јм—җ м Ғмһ¬лҗң кІғл§Ң лӮЁкёҙлӢӨ. 2нҺё м—җм„ң м¶”к°Җн•ң _loaded_at л©”нғҖлҚ°мқҙн„° м»¬лҹјмқҙ м—¬кё°м„ң м“°мқёлӢӨ.

SilverлҘј н•Ёл¶ҖлЎң л°”кҫёл©ҙ Goldк°Җ к№Ём§„лӢӨ#

Gold лӘЁлҚёмқҖ Silver н…Ңмқҙлё”мқҳ м»¬лҹјлӘ…, нғҖмһ…, лӢЁмң„лҘј лҜҝкі  м“ҙлӢӨ. stg_ordersмқҳ order_dateк°Җ DATEлқјлҠ” м „м ңлЎң Goldм—җм„ң лӮ м§ң н•ЁмҲҳлҘј м“°кі  мһҲлҠ”лҚ°, лҲ„кө°к°Җ Silverм—җм„ң м»¬лҹјлӘ…мқ„ ordered_atмңјлЎң л°”кҫёл©ҙ Gold лӘЁлҚёмқҙ м „л¶Җ м—җлҹ¬лҘј лұүлҠ”лӢӨ.

м»¬лҹјмқ„ м¶”к°Җн•ҳлҠ” кұҙ кҙңм°®лӢӨ. кё°мЎҙ м»¬лҹјмқҳ мқҙлҰ„мқҙлӮҳ нғҖмһ…мқ„ л°”кҫёлҠ” кІҢ мң„н—ҳн•ҳлӢӨ. dbtмқҳ ref() н•ЁмҲҳк°Җ мқҳмЎҙ кҙҖкі„лҘј м¶”м Ғн•ҳлӢҲк№Ң м–ҙл””к°Җ мҳҒн–Ҙ л°ӣлҠ”м§ҖлҠ” м•Ң мҲҳ мһҲлӢӨ.

мӢӨл¬ҙ м°ёкі : Airflowм—җм„ң dbt мӢӨн–ү#

Airflowм—җм„ң dbtлҘј мӢӨн–үн•ҳлҠ” л°©лІ•мқҖ м—¬лҹ¬ к°Җм§ҖлӢӨ. к°ҖмһҘ к°„лӢЁн•ң кұҙ BashOperatorлЎң dbt runмқ„ нҳём¶ңн•ҳлҠ” кІғмқҙкі , лҚ” м •көҗн•ҳкІҢ н•ҳл Өл©ҙ cosmos лқјмқҙлёҢлҹ¬лҰ¬лҘј м“ҙлӢӨ.

from airflow import DAG
from airflow.operators.bash import BashOperator
from datetime import datetime

with DAG(
    dag_id='silver_transformation',
    schedule='0 6 * * *',
    start_date=datetime(2026, 1, 1),
    catchup=False,
) as dag:

    # Bronze м Ғмһ¬ мҷ„лЈҢлҘј кё°лӢӨлҰ° л’Ө Silver ліҖнҷҳ мӢӨн–ү
    run_staging = BashOperator(
        task_id='dbt_run_staging',
        bash_command='cd /opt/dbt/jaffle_shop && dbt run --select staging',
    )

    # dbt testлЎң Silver лҚ°мқҙн„° н’Ҳм§Ҳ кІҖмҰқ
    test_staging = BashOperator(
        task_id='dbt_test_staging',
        bash_command='cd /opt/dbt/jaffle_shop && dbt test --select staging',
    )

    run_staging >> test_staging

dbt run лӢӨмқҢм—җ dbt testлҘј кұҙлӢӨ. Silver ліҖнҷҳмқҙ лҒқлӮҳл©ҙ л°”лЎң н’Ҳм§Ҳ кІҖмҰқмқ„ лҸҢлҰ°лӢӨ. н…ҢмҠӨнҠёк°Җ мӢӨнҢЁн•ҳл©ҙ Gold ліҖнҷҳмңјлЎң л„ҳм–ҙк°Җм§Җ м•ҠлҠ”лӢӨ. л¶Ҳлҹү лҚ°мқҙн„°к°Җ Goldк№Ңм§Җ мҳ¬лқјк°ҖлҠ” кұё л§үлҠ” кө¬мЎ°лӢӨ.

cosmos лқјмқҙлёҢлҹ¬лҰ¬лҘј м“°л©ҙ dbt лӘЁлҚё н•ҳлӮҳн•ҳлӮҳлҘј Airflow нғңмҠӨнҒ¬лЎң л¶„лҰ¬н•  мҲҳ мһҲлӢӨ. stg_ordersк°Җ мӢӨнҢЁн•ҙлҸ„ stg_customersлҠ” лҸ…лҰҪм ҒмңјлЎң м„ұкіөн•ңлӢӨ. лӘЁлҚёмқҙ мҲҳмӢӯ к°ңлЎң лҠҳм–ҙлӮҳл©ҙ мқҙ м„ёл¶„нҷ”к°Җ мқҳлҜё мһҲм–ҙм§„лӢӨ.

лӢӨмқҢ кёҖм—җм„ңлҠ” SCD(Slowly Changing Dimension)лҘј лӢӨлЈ¬лӢӨ. кі к°қмқҳ мЈјмҶҢк°Җ л°”лҖҢм—Ҳмқ„ л•Ң кіјкұ° мЈјмҶҢлҘј м–ҙл–»кІҢ ліҙмЎҙн•ҳлҠ”к°Җ. Type 1, 2, 3мқҳ м°ЁмқҙмҷҖ м„ нғқ кё°мӨҖ.
