구글 스프레드시트 대신 터미널로 관리하는 주식 포트폴리오

4월 26, 2026 · 2 분 · Junho Lee | 번역:
  • En
목차
  • 내 컴퓨터에 저장하는 투자 데이터
  • 인공지능 비서로 진화하는 포트폴리오
  • 스크립트 연동과 확장성

해외 주식 포트폴리오를 관리하려고 구글 스프레드시트를 열 때마다 한숨이 나온다. 수동 입력은 귀찮고 API 연동은 매번 말썽을 부리기 때문이다. firma는 이런 반복적인 작업을 터미널 환경으로 옮겨온 도구다.

내 컴퓨터에 저장하는 투자 데이터#

자산 정보는 민감한 영역이다. 외부 서버에 의존하는 대신 로컬 SQLite 파일에 모든 기록을 보관하여 보안 문제를 해결했다. ~/.firma/firma.db 파일 하나로 모든 수치가 관리되니 유출 걱정도 줄어든다.

Finnhub API를 통해 시장 가격을 직접 가져오기에 중간에 거치는 과정이 없다. firma show portfolio 명령어를 입력하면 현재 손익과 비중이 화면에 나타난다. 간결하고 명확한 구성이다.

매수와 매도 기록은 물론 배당금과 세금 내역까지 firma add txn으로 관리한다. 정보가 쌓일수록 월별 순자산 추이나 현금흐름의 정확도가 높아진다. firma report 기능은 시각적 피드백을 주기에 충분하다.

인공지능 비서로 진화하는 포트폴리오#

Claude Desktop에 MCP 서버로 등록하면 운용 도구는 지능형 비서로 진화한다. firma mcp install 실행 한 번으로 모든 설정이 마무리된다. 자연어로 현재 비중을 묻거나 종목 뉴스를 분석해달라고 요청할 수 있다.

AI가 로컬 DB를 직접 조회해 답변을 내놓는다. 이번 달 지출이 왜 높은지 물으면 전월 대비 내역을 분석해준다. 단순한 챗봇을 넘어 데이터베이스를 직접 다루는 에이전트 역할을 수행한다.

매매 내역 입력까지 대화로 처리하니 도구 사용의 진입 장벽이 낮아진다. 기술적 접점이 매끄럽게 이어진다.

스크립트 연동과 확장성#

모든 읽기 명령어에 --json 플래그를 지원한다. 특정 조건에서 알림을 보내거나 커스텀 대시보드를 만드는 작업이 한결 수월하다. firma show earnings 입력을 통해 재무 수치를 파싱하기에도 적합하다.

API 키 설정도 firma config set 명령어로 간단히 해결한다. 툴 자체가 가볍고 의존성이 적어 설치 부담이 없다. 터미널 환경을 선호하는 이들에게 최적화된 구성이다.

복잡한 UI 없이도 자산을 완벽하게 통제하는 경험은 쾌적하다.

핵심만 뽑으면

  • 로컬 SQLite 저장 방식을 채택해 금융 데이터의 프라이버시 문제를 엔지니어링적으로 해결했다.
  • MCP(Model Context Protocol)를 활용해 CLI 도구와 LLM 사이의 데이터 인터페이스를 표준화했다.
  • 단순 조회를 넘어 매매 기록 입력까지 AI에게 위임하는 워크플로우를 구현해 사용자 경험을 개선했다.
