2026년 상반기 AI FOMO를 해결하는 가장 저렴한 에이전트 구축법

5월 28, 2026 · 5 분 · Junho Lee
목차
  • 맥북과 스마트폰을 연결하는 기본 설정
  • 3일 사용 후 시작하는 업무 자동화
  • 비용 대비 최선의 선택지

2026년 상반기 AI FOMO 없애는 가장 가성비 넘치는 방법: Codex 사용하세요 AGENTS.md (or CLAUDE.md) 만들라, Skill 세팅하라, Harness 구성하라. 보통 이런 얘기를 들으면 “해야 하는 건 알겠는데 그걸 할 시간이 없다” 라는 생각이 듭니다. 심지어 Openclaw, Hermes-agent 같은 도구도 있다는데 용어만 듣고 할 엄두가 안 나죠. 하지만 2026년 상반기 기준 에이전트를 가장 쉽고 저렴하게 사용할 수 있는 방법이 있습니다. Ruby on Rails 언어를 만든 사람 등 최고 레벨의 개발자를 포함해 주요 AI 관련 인물도 극찬하고 있는 Codex 를 “그냥” 사용하는 것입니다. 그래도 감이 잘 안오시는 분들을 위해 아래에 그동안 AI 관련 아무것도 하지 않으셨더라도 가장 크게 효과를 볼 수 있는 꿀팁을 상세하게 적었습니다. ① 준비물은 이게 전부입니다 1) 맥북 + Chrome 브라우저 + Codex 데스크톱 앱 (+ Amphetamine 데스크톱 앱) 2) ChatGPT 모바일 앱 (iOS·Android 모두 가능) 3) ChatGPT Plus 이상 구독 ② Chrome 웹스토어에서 Codex 확장 프로그램을 설치하세요 검색하고 추가하면 끝입니다. 이걸 설치해야 Codex 가 로그인된 웹사이트 안에서 작업할 수 있거든요. ③ Codex 데스크톱 앱에서 플러그인을 설치하세요 좌측 탭 상단 ‘플러그인’ 클릭 후 다음 7개를 설치합니다. Computer Use, Chrome, Spreadsheets, Presentations, Gmail (혹은 Outlook Email), Google Calendar, Google Drive. 일단 이것만 깔아두면 왠만한 업무 자동화가 가능합니다. ④ Codex 데스크톱 앱에서 작업 모드와 권한을 바꿉니다 ‘설정’ → 일반 → 작업 모드를 코딩용으로 전환하세요. 권한은 기본 권한, 자동 검토, 전체 접근 권한 모두 on 하면 좋아요. 파일 삭제가 두렵다면 자동 검토까지만 켜도 됩니다. ⑤ Codex 데스크톱 앱에서 Computer Use 를 활성화하세요 ‘설정’ → 컴퓨터 사용 → 모든 앱, Google Chrome, 잠금 사용 세 가지를 모두 on 하세요. 이걸 켜야 Codex 가 맥북 화면을 보고 앱을 직접 조작할 수 있습니다. ⑥ Codex 데스크톱 앱에서 핸드폰과 맥북을 연결합니다 ‘설정’ → 연결 → ‘이 Mac 제어’에서 추가 버튼을 누르고 핸드폰을 연결하세요. “이 기기가 검색되고 제어되도록 허용"과 “이 Mac 잠자기 방지"를 둘 다 켜야 합니다. 특히 잠자기 방지를 빼먹으면 외출 중에 Codex 가 멈추니까 주의가 필요해요. ⑦ Chrome 에 자주 쓰는 사이트를 미리 로그인해 둡니다 맥북에서 크롬 브라우저를 열고 업무에 자주 사용하는 사이트에 로그인을 미리 해두세요. 그리고 크롬 브라우저 상단 퍼즐 아이콘 → 전체 액세스에 Codex 가 표시되어 있는지도 확인하세요. 이렇게 하면 Codex 가 모든 웹 작업을 쉽게 할 수 있습니다. ⑧ 핸드폰에서 Codex 탭을 열고 작업을 시작합니다 ChatGPT 모바일 앱 → 좌측 메뉴 아이콘 클릭 → Codex 탭에 접속합니다. 그러면 핸드폰에서도 쉽게 AI 에이전트와 작업할 수 있습니다. 맥북의 Amphetamine 앱을 함께 실행시키면 맥북을 닫고도 계속 사용할 수 있습니다. 다만 이 경우 맥북을 닫아도 절전 상태로 전환이 안되니 주의하시기 바랍니다. 예를 들어 웹 작업이 필요하면 @chrome, 컴퓨터 파일이나 프로그램이 필요하면 @computer 를 붙여서 명령하면 됩니다. 타이핑 입력이 귀찮다면 보이스 모드로 말로 입력하세요. ⑨ 시범 삼아 3일 정도 사용하고 나서 자동화를 세팅합니다 3일 정도 직접 써보면 매일 AI 에게 반복적으로 요청하는 작업을 알게 됩니다. 이 때, Codex 탭에서 “매일 몇 시에 이 작업을 진행해서 결과를 가져다줘"라고 명령하면 ‘자동화’ 탭에 바로 저장됩니다. 이벤트 트리거 방식(“이런 일이 생겼을 때 실행해줘”)도 되는데 조건을 조금 구체적으로 써줘야 합니다. 결과물 품질이 아쉬우면 기억날 때마다 Codex 탭에 추가로 말해두세요. 그러면 점점 결과물의 수준이 좋아집니다. ⚠️ 주의 사항으로 Meta, LinkedIn 등 SNS 관련 작업은 시키지 마세요. 계정이 삭제될 수 있습니다. 이게 끝입니다. 월 3만원 ChatGPT Plus 구독 하나로도 이 모든 게 됩니다. GPT-5.5 가 가장 성능이 좋지만 가격이 저렴한 GPT-5.4 나 GPT-5.3-Codex 도 글쓰기가 아닌 에이전트 작업으로는 충분히 쓸만해요. 이 외에도 현존하는 가장 저렴한 AI 인 딥시크의 API 를 연결하는 방법도 있기는 한데 트릭이 좀 필요합니다. 그래서 부차적인 고민하지 말고 3만원으로 한 달만 Codex 를 사용해보세요. 이 시대에 AI 에이전트를 이렇게 간편하고 저렴하게 쓸 수 있는 방법은 없습니다.

자동화 도입을 고민하다가 설정 단계에서 지쳐 포기하는 엔지니어가 많다. AGENTS.md 파일을 만들 시간은 늘 부족하다. 대안이 있다. 지금 가장 빠르고 현실적인 방법은 Codex 가 제공하는 시스템을 바로 사용하는 방법이다.

맥북과 스마트폰을 연결하는 기본 설정#

스마트폰 하나로 내 맥북을 원격 제어하는 환경은 몇 가지 앱 설치로 끝난다. 맥북에 데스크톱 앱과 확장 프로그램을 설치하고 스마트폰 앱을 연동하면 된다. 준비물은 맥북과 모바일 기기, 구독권이 전부다. 간단하다.

먼저 데스크톱 앱에서 Computer Use 옵션과 필수 플러그인을 활성화한다. Chrome, Spreadsheets, Gmail 등 업무 도구 7개를 설치해 둔다. 설정 메뉴에서 맥북 제어 허용과 잠자기 방지 옵션을 켜지 않으면 외출 시 앱이 멈춘다. 낭패다.

크롬 브라우저에 주로 쓰는 업무 사이트를 로그인해 두면 준비가 끝난다. 모바일 앱의 전용 탭에서 @chrome 이나 @computer 명령어로 원격 제어를 시작한다. 맥북 화면을 닫고 구동하려면 Amphetamine 앱을 활용하되 발열을 고려해야 한다. 주의가 필요하다.

3일 사용 후 시작하는 업무 자동화#

업무 자동화는 3일 동안 반복 작업을 직접 시켜본 뒤 등록하는 방식이 안전하다. 며칠간 사용 패턴이 쌓이면 자주 요청하는 작업이 눈에 들어온다. 이때 전용 탭에 주기적인 작업 수행을 명령하면 자동화 설정이 완료된다. 효율이 높다.

이벤트 트리거를 걸 때는 작동 조건을 명확하게 텍스트로 적어야 오류가 없다. 결과물 품질이 아쉬우면 피드백을 수시로 덧붙여 성능을 개선한다. 다만 계정 차단 위험이 있는 SNS 작업은 시스템에 지시하지 않는 규칙을 세운다. 계정이 날아간다.

비용 대비 최선의 선택지#

월 3만 원 수준의 구독료만 지불하면 추가 비용 없이 고성능 시스템을 굴린다. GPT-5.5 모델이 가장 정교하지만 가벼운 작업은 GPT-5.4 버전으로도 원활하게 돌아간다. 다른 저가형 API를 연동하는 복잡한 트릭을 고민할 시간에 이 구성을 먼저 써보는 편이 이득이다. 시간이 절약된다.

자동화 기술을 어렵게만 보던 시선이 싹 사라진다. 이제 손안의 스마트폰으로 맥북을 깨워 일을 시킨다. 충분하다.

핵심만 뽑으면

  • AGENTS.md 작성이나 복잡한 프레임워크 학습 없이 Codex 데스크톱 앱과 모바일 연동만으로 실용적인 에이전트 환경을 즉시 구축한다.
  • Computer Use 기능과 주요 플러그인 7개를 활성화하고, 잠자기 방지 옵션을 켜두면 외출 중에도 스마트폰으로 맥북을 원격 제어할 수 있다.
  • 3일 동안 직접 구동하며 반복 요청 패턴을 파악한 뒤 텍스트 기반 트리거를 등록하는 방식으로 안정적인 업무 자동화를 완성한다.
